Requerimientos de Minecraft

Información sobre hardware y software para ejecutar Minecraft en PC.

Requerimientos mínimos

Componentes y especificaciones

  • Sistema operativo: Windows 7 (64-bit) o posterior; macOS Mojave (10.14.5) o superior; Linux moderno de 64-bit
  • Procesador (CPU): Intel Core i3-3210 o AMD A8-7600
  • Memoria RAM: 4 GB
  • Tarjeta gráfica (GPU): Intel HD Graphics 4000 / AMD Radeon R5 o dedicada modesta (GeForce 400 series / Radeon HD 7000) con soporte OpenGL 4.4
  • Almacenamiento: Al menos 1 GB libre en HDD
  • Resolución: 1024×768 o superior

Requerimientos recomendados

Componentes y especificaciones

  1. Sistema operativo: Windows 10/11 (64-bit); versiones recientes de macOS / Linux
  2. Procesador (CPU): Intel Core i5-4690 o equivalente AMD
  3. Memoria RAM: 8 GB
  4. Tarjeta gráfica (GPU): GeForce 700 series / Radeon R7 / RX 200 series con soporte OpenGL 4.5
  5. Almacenamiento: 4 GB o más, preferiblemente en SSD
  6. Resolución: 1920×1080 (FULL HD)
